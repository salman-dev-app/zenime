### Prerequisites
- A modern web browser
- (Optional) Node.js 18+ for local dev server

### Setup & Run
```bash
# Clone this repository
git clone https://github.com/salman-dev-app/zenime.git
cd zenime

# Open in browser directly
open index.html

# Or use a local dev server
npx serve .
```
